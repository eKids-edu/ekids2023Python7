# EPAM програма eKids 2023, група Python-7

Створення Телеграм бота, що зможе працювати зі списком товарів.

Що може бот:

* додавати товар
* видаляти товар
* виводити список товарів
* редагувати властивості окремого товару
* робити аналітику по товарам на складі
